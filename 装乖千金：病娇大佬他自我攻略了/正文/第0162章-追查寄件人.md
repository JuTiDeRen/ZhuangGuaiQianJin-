# 第0162章 追查寄件人
上午九点，阳光透过书房的百叶窗，在地板上投下一道道明亮的光影。
沈知意坐在书桌前，面前摆着那个黑色盒子。
一夜未眠，她的眼下有着淡淡的青黑，但眼神却异常清醒。她在等——等一个合适的时机，等沈家老宅的其他人都出门后，再开始她的调查。
现在，是时候了。
她先拿起手机，查看了快递信息。
快递单号显示，包裹是通过一家名为"顺风达"的同城快递公司寄送的。寄件人信息栏只填了一个化名："顾先生"。联系电话是空号。
沈知意皱了皱眉。
同城快递。这意味着寄件人就在京城。
而"顾"这个姓氏——她母亲的姓氏。
是巧合，还是故意为之？
她拨通了顺风达的客服电话，以"寄件人填错信息需要核实"为由，询问这单快递的具体收件地点。
客服的声音带着程式化的礼貌："您好，这单快递是从城西的望山分部发出的，具体寄件地址……抱歉，寄件人要求保密，我们无法提供。"
"望山分部？"沈知意追问，"那收件员是谁？我可以直接联系他吗？"
"这个……我需要查询一下。"
电话那头传来键盘敲击的声音，片刻后，客服回答："收件员是王强，工号0473。但按规定，我们不能透露员工的私人联系方式……"
沈知意打断了她："理解。我会直接去望山分部找他。"
挂断电话，她起身换了身衣服——简单的白色针织衫和牛仔裤，头发扎成马尾，整个人看起来乖巧而普通。
这是她最擅长的伪装。
望山分部位于京城西郊，一个老旧的物流园区内。沈知意开车抵达时，已经是上午十点。
分拣中心里一片忙碌，快递堆积如山，工作人员穿梭其中。
她在前台报出了王强的工号，很快就见到了这个男人——三十出头，皮肤黝黑，一双眼睛在看到她时闪过明显的困惑。
"你是那个……沈家老宅的收件人？"王强挠了挠头，"怎么了，包裹有问题？"
"没有问题。"沈知意微微一笑，声音轻柔，"我只是想问问，那个寄件人……你还记得他长什么样吗？"
王强的表情僵了一瞬。
那一瞬很短暂，不到半秒。但沈知意的瞳孔却微微收缩——她捕捉到了。
困惑。还有隐瞒。
"就是个普通男人，四十多岁，戴着口罩和帽子，看不太清脸。"王强的语速比平时快了一些，"他直接来站点寄的，付了现金，没说什么。"
沈知意盯着他的眼睛。
微表情洞察，启动。
第一次。
王强的视线在回答时下意识地向右上方飘移——这是大脑在调用想象区域，而不是记忆区域。他在编造，而不是回忆。
他的右手无意识地握紧了衣角——紧张的表现。
还有，当他说"没说什么"的时候，嘴角微微向下撇了一下——这是典型的隐瞒信号。
他在说谎。
"王先生，"沈知意的声音依然轻柔，但眼神却锐利了几分，"你确定……他什么都没说？"
王强的喉结滚动了一下。
"我……"
"或者我应该换个问法，"沈知意向前一步，压低了声音，"他给了你多少钱，让你保守秘密？"
王强的脸色瞬间变了。
"你、你在说什么？我不明白……"
"一万？两万？"沈知意继续说道，语气平静得像是在谈论天气，"让我猜猜，他给你钱的时候，应该还说了什么威胁的话吧？比如，如果你泄露他的信息，会有什么后果？"
王强的额头开始冒汗。
沈知意知道自己猜对了。
"我、我真的不能说……"王强的声音开始发抖，"他说了，如果我告诉别人，我、我就完了……"
"那如果我说，你不告诉我，你现在就完了呢？"沈知意的嘴角依然挂着微笑，但那笑意却不达眼底。
她从包里拿出一张名片，递到王强面前。
"沈氏集团，沈知意。"她的声音很轻，却带着不容置疑的压迫感，"你收的那个人的钱，我三倍给你。但你必须把你知道的一切都告诉我。"
王强盯着那张名片，脸色苍白。
沈氏集团。京城顶级的豪门。
他一个小小的快递员，哪里惹得起？
"我、我说……"王强的声音低了下来，"那人确实给了我一万块，让我保密。但他没告诉我他是谁，真的。他戴着口罩和墨镜，看不清脸。"
"身高？体型？"
"一米七五左右，不胖不瘦。说话声音……有点沙哑。"
"他什么时候来的？"
"前天晚上，快关门的时候。大概……九点多。"
沈知意默默记下这些信息。
"还有，"王强犹豫了一下，"他让我转交一句话。"
"什么话？"
"他说：'告诉她，火漆印章上的图案，是答案的一半。'"
沈知意的心猛地一跳。
火漆印章。
她立刻从包里拿出那个黑色盒子，指着封口处的火漆印章："是这个吗？"
王强看了一眼，点点头："对，他特意让我注意这个。说如果收件人问起，就把这句话带给她。"
沈知意盯着那枚暗红色的火漆印章。
图案隐约像是某种家族徽记——云纹缠绕着一枝梅花，梅花的中央是一个篆体的"沈"字。
沈家旧徽记。
她认得这个图案。在沈氏祖宅的某些老物件上，在沈家的族谱里，她都曾见过。
这是沈家三十年前使用的旧徽记，后来随着家族企业现代化，换成了更简洁的LOGO，这个旧徽记就渐渐不再使用了。
只有沈家内部的人，才知道这个图案的存在。
寄件人用这个徽记，是想告诉她什么？
他是沈家的人？还是故意用沈家的徽记来暗示什么？
"谢谢。"沈知意收回盒子，从包里拿出一沓现金——她出门时特意准备的，刚好三万块。
"这是你的报酬。"她将钱递给王强，"还有，如果你再想起什么，可以打这个电话找我。"
她留下一张私人名片，转身离开。
王强握着那沓钱，愣在原地。
走出物流园区，沈知意坐回车里，却没有立刻发动。
她盯着那枚火漆印章，大脑飞速运转。
包裹来自京城本地。
寄件人知道沈家旧徽记。
他还知道她会来追查，提前准备了那句"火漆印章上的图案，是答案的一半"。
这一切，都像是一个精心设计的局。
而她，已经踏了进去。
手机突然震动，是陆砚辞发来的消息："中午有空吗？一起吃饭。"
沈知意看着屏幕上的文字，手指悬在键盘上方，迟迟没有落下。
她想起了字条上的警告——"不要告诉任何人，包括陆砚辞"。
她也想起了照片里母亲温柔的笑脸。
还有火漆印章上那个篆体的"沈"字。
如果母亲的死真的与沈家内部有关，那她就必须更加小心。在查清真相之前，她不能相信任何人。
哪怕那个人是陆砚辞。
她深吸一口气，回复："今天有点忙，改天吧。"
发送。
然后将手机放到一旁，发动汽车。
下一站——沈氏祖宅。
她要去东厢房，看看那里究竟藏着什么秘密。
车子驶入主路，沈知意从后视镜里看了一眼。
一辆黑色轿车不知何时跟在了后面，保持着不远不近的距离。
是巧合，还是……
她想起了那个戴着口罩和帽子的寄件人。
以及火漆印章上那个篆体的"沈"字。
（本章完）
