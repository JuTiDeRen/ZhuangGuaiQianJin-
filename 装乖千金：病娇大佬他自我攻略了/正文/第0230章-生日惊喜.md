# 第0230章：生日惊喜

车没有开回沈家老宅。

沈知意看着窗外越来越熟悉的路线，心里泛起一丝疑惑。

"这不是回家的路。"

"嗯。"陆砚辞握着方向盘，目光专注地看着前方，唇角却悄悄扬起，"说了有别的安排。"

"你到底要带我去哪儿？"

"到了你就知道了。"

沈知意偏头看他，试图从他的侧脸读出什么。但陆砚辞的表情很平静，甚至可以说是……享受？

他似乎在享受她的好奇和期待。

"你就不能透露一点？"她放软了声音，带着几分撒娇的意味。

陆砚辞终于转过头看了她一眼，眼底含着笑："小公主这是在跟我装乖套话？"

被看穿了。

沈知意脸一红，别过头去："才没有。"

"有没有都没用。"陆砚辞的心情似乎很好，"今天这关，你过不了。"

沈知意："……"

这个人，真是越来越难对付了。

车继续前行，穿过城市的繁华地段，驶入一条沈知意不太熟悉的路。

又过了十分钟，车在一栋摩天大楼前停下。

这是京城最高的建筑——云巅大厦，共108层，顶层是只对顶级VIP开放的私人会所。

"来这里做什么？"沈知意更疑惑了。

陆砚辞没有回答，只是下车，绕到副驾驶，为她打开车门。

"跟我来。"

他牵起她的手，十指相扣，带着她走进大厦。

专属电梯直达顶层。

当电梯门打开的那一刻，沈知意愣住了。

眼前是一片巨大的玻璃穹顶，透过玻璃可以看到满天的繁星。穹顶下布置得如同梦幻仙境——

无数盏暖黄色的灯串悬挂在空中，像是倒悬的星河。

地面上铺满了白色的玫瑰花瓣，踩上去柔软得像是在云端。

正中央是一张精致的餐桌，上面摆着烛台和鲜花。

而最让她震惊的是，玻璃穹顶外，是整个京城的夜景。

万家灯火在脚下铺展开来，像是一片璀璨的灯海。

"这是……"她的声音有些发抖。

"云巅私人天文台。"陆砚辞从身后环住她的腰，下巴抵在她的肩窝里，"我包下了整个顶层。"

沈知意转身看他，眼眶有些发热。

"你什么时候准备的这些？"

"很久以前就开始准备了。"陆砚辞低头看着她，眼神深邃，"你说你喜欢看星星，但京城的污染太重，看不到几颗。"

沈知意愣住了。

她确实说过这样的话，那是很久以前，在两人还没确定关系的时候。

那时候她随口提了一句，没想到他竟然记得。

"这里的天文望远镜可以看到土星环。"陆砚辞牵着她走向角落里的专业设备，"要不要看看？"

沈知意点点头，任由他带着自己来到望远镜前。

陆砚辞调试着设备，动作熟练而专注。沈知意看着他的侧脸，忽然觉得，这个人好像一直在用他自己的方式，记住她说的每一句话。

"好了。"他退开一步，"看看。"

沈知意凑近目镜，下一秒，她屏住了呼吸。

她看到了土星。

那个带着美丽光环的行星，此刻清晰地呈现在她的视野里，像是一颗悬浮在宇宙中的宝石。

"好美……"她喃喃道。

"是很美。"陆砚辞的声音从身后传来，但似乎不是在说土星。

沈知意抬起头，发现他正在看着自己。

"我说的是土星。"她小声说。

"我说的是你。"陆砚辞把她拉进怀里，"看星星的时候，眼睛里有光。"

沈知意的脸瞬间红了。

这个人，真是……越来越会说情话了。

两人在天文台里待了两个小时。

他们一起看了土星、木星，还有遥远的仙女座星系。

陆砚辞还准备了一场小型的灯光秀——玻璃穹顶外，无人机编队飞上夜空，组成了"生日快乐"和一颗心形图案。

沈知意站在玻璃前，看着那璀璨的灯光，眼泪不知不觉地滑落。

"怎么哭了？"陆砚辞伸手擦掉她的眼泪，声音里带着几分紧张，"不喜欢？"

"不是……"沈知意摇摇头，"是太喜欢了。"

她转过身，主动抱住了他。

"谢谢你，砚辞。"

这是她第一次主动抱他。

陆砚辞的身体僵硬了一瞬，随即收紧了手臂，把她紧紧箍在怀里。

"傻瓜。"他的声音有些沙哑，"你想要的，我都会给你。"

他在心里默默加了一句：包括我自己。

但他没有说出口。

那个最重要的承诺，要留到最重要的时候。

离开云巅大厦时，已经是深夜。

沈知意坐在副驾驶上，手里还攥着刚才在天文台摘的一朵白玫瑰。

"今天开心吗？"陆砚辞问。

"很开心。"沈知意点头，"这是我过得最开心的一个生日。"

"只是最开心的？"

"嗯？"

陆砚辞转过头看她，眼神里带着几分神秘的笑意："我说过了，今天只是开始。"

沈知意的心跳漏了一拍。

又是这句话。

"你到底……还准备了什么？"她小心翼翼地问。

陆砚辞没有直接回答，只是伸手揉了揉她的头发。

"平安夜。"

他的声音很轻，却像是一颗石子投入平静的湖面，在沈知意的心里激起层层涟漪。

"平安夜……会发生什么？"

"到时候你就知道了。"陆砚辞收回手，重新握住方向盘，"不过我可以告诉你，那会比今天更让你难忘。"

沈知意低下头，看着手里的白玫瑰，心跳快得像是要从胸腔里跳出来。

平安夜。

还有六天。

那到底会是什么？

求婚吗？

还是别的什么更大的惊喜？

她偷偷看了陆砚辞一眼，发现他的唇角带着笑，似乎心情很好。

"又在偷看我？"他忽然开口。

"我才没有！"沈知意慌乱地转过头。

"没关系。"陆砚辞笑出了声，"想看就看，我不收费。"

沈知意："……"

这个人，真是自恋得让人无话可说。

但不知道为什么，她的心里却甜得像是要化开。

六天。

还有六天。

她闭上眼睛，靠在座椅上，嘴角忍不住上扬。

不管平安夜会发生什么，她都无比期待。

因为，那一定是这个世界上，最美好的惊喜。
