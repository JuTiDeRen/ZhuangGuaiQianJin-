# 第160章 新的开始

深夜十一点，沈知意回到了沈家老宅。

客厅里灯火通明，沈父沈明德坐在沙发上，手中夹着一支雪茄，神情凝重。看到沈知意进来，他抬起头，眼中闪过一丝复杂的情绪。

"回来了。"

"爸。"沈知意在他对面坐下，"您怎么还没睡？"

"睡不着。"沈明德叹了口气，"今天发生的事太多了。"

他顿了顿，继续说："顾长渊被劫走的事，我已经听说了。"

沈知意没有说话，等待着他的下文。

"知意，"沈明德看着她，眼神中带着几分愧疚，"这些年，是爸对不起你。"

"爸，"沈知意打断他，"过去的事就不要提了。"

"不，我必须说。"沈明德的声音有些沙哑，"六年前，如果我肯相信你，你就不会被流放。如果我能多关心你一些，你也不会吃那么多苦。"

他低下头，声音变得低沉："是我这个做父亲的失职。"

沈知意看着眼前这个男人。六年前，他是那个相信沈清瑶而不相信她的父亲。六年后，他坐在这里，向她道歉。

"爸，"她说，"我现在不需要道歉。我需要的是真相。"

沈明德抬起头："什么真相？"

"我母亲，"沈知意直视着他的眼睛，"她真的是病死的吗？"

沈明德的脸色瞬间变了。

他的手微微颤抖，雪茄的灰烬落在地毯上，他却浑然不觉。

"你……你怎么会这么问？"

"顾长渊告诉我，母亲不是病死的。"沈知意的声音很平静，"他说，母亲是被人害死的。"

沈明德沉默了。

良久，他开口了："你母亲……确实是病逝的。"

"真的吗？"沈知意追问。

沈明德避开了她的目光："医院有诊断记录，你可以去查。"

"诊断记录可以伪造。"沈知意说，"爸，如果您知道什么，请告诉我。"

沈明德张了张嘴，似乎想要说什么，但最终还是摇了摇头："知意，有些事情，不知道比知道更好。"

"这是什么意思？"

"我的意思是，"沈明德站起身，"你母亲的死已经过去了这么多年，何必再追究？你现在已经是沈氏的董事长了，应该往前看，而不是纠结于过去。"

沈知意看着他，心中涌起一股失望。

"爸，"她说，"您不愿意说，我不逼您。但我自己会查清楚。"

她站起身，向楼上走去。

"知意！"沈明德在身后叫住她。

她停下脚步，但没有回头。

"不管你要做什么，"沈明德的声音有些疲惫，"都要小心。有些人……不是你想象的那么简单。"

沈知意没有回答，继续向楼上走去。

回到房间，她靠在门上，闭上眼睛。

父亲的反应证实了她的猜测——母亲的死，确实另有隐情。

而且，父亲知道些什么，却不愿意告诉她。

手机突然响了。

是一个陌生号码发来的短信：

"沈小姐，有一份您的快递放在老宅门口，请注意查收。"

沈知意皱了皱眉。

这么晚了，谁会给她送快递？

她走下楼，来到门口。果然，地上放着一个黑色的盒子，没有寄件人信息，只有她的名字。

她蹲下身，将盒子拿起来。

盒子不重，但触感有些奇怪，像是里面装着什么金属制品。

她回到房间，将盒子放在桌上，小心翼翼地打开。

里面是一张照片，一把钥匙，还有一张字条。

照片上是年轻时的母亲，站在一栋老宅前，笑得温柔而灿烂。照片背面写着一行字："这是你母亲生前最后一张照片，拍摄于她去世前一周。"

钥匙是古铜色的，样式很老旧，上面刻着一行小字："沈氏祖宅，东厢房。"

字条上的内容让沈知意的瞳孔猛地收缩：

"想知道你亲生母亲是怎么死的吗？去沈氏祖宅的东厢房，那里有你要的答案。"

"记住，不要告诉任何人，包括陆砚辞。"

"因为，你母亲的死，与陆家有关。"

沈知意的手微微颤抖。

她母亲的死，与陆家有关？

这是什么意思？

她的脑海中闪过陆砚辞的脸，闪过他说过的那些话。

他说，不管发生什么，他都会站在她这边。

他说，他已经等了她六年。

他说，他喜欢她。

这一切，都是真的吗？还是……另有隐情？

沈知意将照片、钥匙和字条重新放回盒子里，锁进了抽屉。

她走到窗前，看着窗外的夜色。

城市灯火阑珊，车水马龙。

而在这繁华的背后，隐藏着多少不为人知的秘密？

母亲的死。

陆家的关系。

顾长渊的阴谋。

还有那个神秘人，是谁？

沈知意深吸一口气，将思绪压下。

不管前面有多少谜团，她都会一一解开。

因为这是她的命运。

也是她的使命。

手机又响了，这次是陆砚辞发来的消息：

"到家了吗？"

沈知意看着屏幕，良久，回复道："到了。"

"早点休息，明天我来接你。"

"好。"

她放下手机，嘴角勾起一抹意味深长的笑。

陆砚辞，如果你真的没有隐瞒我什么，那就证明给我看。

如果你也是局中人……

那就别怪我不念旧情了。

沈知意关上灯，躺到床上。

黑暗中，她的眼睛依然明亮。

属于她的战斗，还远未结束。

而那些隐藏在暗处的敌人，终将一个个浮出水面。

她等着那一天的到来。

与此同时，城市另一端。

一栋废弃的工厂内，顾长渊站在窗前，看着远处的城市灯火。

他的身后，站着一个身穿黑色风衣的男人。

"计划进行得很顺利。"男人说，"沈知意已经收到那个盒子了。"

"很好。"顾长渊的嘴角勾起一抹意味深长的笑，"让她去查吧。查得越深，她就会发现，真相远比她想象的可怕。"

"陆氏那边……"

"陆家？"顾长渊冷笑，"陆家才是这一切的开始。沈知意以为她赢了，殊不知，她才刚刚踏入棋局。"

他转过身，看着窗外的夜色："告诉那边的人，可以开始下一步了。"

"是。"

男人转身离去，脚步声在空旷的工厂内回响。

顾长渊独自站在黑暗中，眼神阴鸷而疯狂。

"沈知意，"他喃喃自语，"游戏，才刚刚开始。"

第2卷·博弈与升温 完
