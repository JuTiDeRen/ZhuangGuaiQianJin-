# 第0037章 项目危机初现
晨光透过落地窗洒进陆氏集团顶层办公室，在深色木地板上投下斑驳的光影。
沈知意坐在会客区的真皮沙发上，手里捧着一杯还冒着热气的拿铁。对面，陆砚辞正低头看着手中的文件，眉头微微蹙起。
"裴靳的调查结果。"他将一份报告推到沈知意面前，"那条匿名威胁短信，来源查到了。"
沈知意放下咖啡杯，接过文件。
报告上清晰地显示：短信发送号码经过多层虚拟转接，最终指向一家境外通讯服务商。而支付这笔服务费用的账户，与顾长渊名下的一家空壳公司有关联。
"果然是他。"沈知意眸色微沉。
"不只是他。"陆砚辞身体前倾，声音压低，"裴靳还发现，王经理最近在频繁接触几个我们的核心供应商。"
沈知意心头一紧："他想干什么？"
"暂时不清楚具体计划。"陆砚辞的目光变得深邃，"但肯定是冲着你来的。"
正说着，陆砚辞办公桌上的内线电话突然响起。
他按下免提键，助理的声音传了出来："陆总，沈氏集团那边打来电话，说是有紧急情况需要沈经理立刻回去处理。"
沈知意和陆砚辞对视一眼。
"看来他们已经开始行动了。"陆砚辞站起身，拿起外套，"我送你过去。"
"不用。"沈知意也站了起来，"你去了，他们会说我又要靠男人。"
陆砚辞挑了挑眉，嘴角浮起一抹似笑非笑的弧度："那你需要我的时候，随时打电话。"
"好。"
沈知意走出陆氏集团大楼时，天空已经阴沉下来，乌云密布，像是要下雨的样子。
她打了一辆出租车，刚坐进后座，手机就响了。
是供应商A的负责人。
"沈经理，很抱歉，我们公司内部有些调整，供货可能需要暂停一段时间。"
沈知意瞳孔骤缩："什么调整？合同上写得清清楚楚，你们单方面违约是要赔偿的。"
"这个……我们也无奈，具体情况您可以问王经理，他知道的。"
电话被挂断了。
沈知意的手指攥紧手机，指节发白。她立刻拨通供应商B的电话，响了很久，无人接听。再打供应商C，直接被挂断。
三个核心供应商，同时断供。
车窗外，京北的街道依然车水马龙，行人匆匆。但她却感觉整个世界都在离她远去。一股寒意从脊背升起，她的手指不受控制地微微颤抖。而她，是项目负责人。
项目会在她手中夭折。她回到沈家后的所有努力，都将付诸东流。
一瞬间，她甚至开始怀疑自己：我真的能赢吗？顾长渊那样的对手，我真的能斗得过吗？
司机从后视镜看了她一眼："姑娘，你脸色不太好，没事吧？"
沈知意深吸一口气，强迫自己冷静下来。
"没事，开快点。"
她把手机屏幕按在胸口，闭上眼睛。不能慌，不能乱。就算全世界都站在她的对立面，她也要撑住。
车子在沈氏集团门口停下，沈知意睁开眼睛，眼底的慌乱已经被强行压下，取而代之的是一种近乎冷漠的平静。
她推开车门，走进大楼。
刚走进市场部，就感受到一股不同寻常的紧张气氛。同事们三五成群地聚在一起，窃窃私语，看到她进来，立刻散开，眼神躲闪。
"沈经理！"李主管急匆匆地迎了上来，脸色凝重，"出大事了。"
"什么事？"
"刚刚接到通知，我们的三个核心供应商同时宣布暂停供货。"李主管的声音压得很低，"说是合同条款需要重新商议。"
沈知意瞳孔微缩。
这三个供应商是陆氏沈氏合作项目的核心支撑，一旦断供，整个项目都要停摆。
"什么时候的事？"
"就在半小时前，几乎是同时。"李主管的脸色很难看，"这明显是有人故意安排的。"
沈知意深吸一口气，强迫自己冷静下来。
她快步走向自己的办公室，一边走一边掏出手机。屏幕上显示着三条未读消息，都来自不同的供应商负责人。
内容大同小异：很抱歉，由于公司内部调整，需要暂停供货，具体恢复时间待定。
"王经理呢？"沈知意问跟进来的李主管。
"他一早就请假了，说是身体不舒服。"
身体不舒服？
沈知意冷笑一声。这个时间点请假，未免也太巧合了。
"召集所有项目组成员，十五分钟后开会。"沈知意坐到办公桌前，打开电脑，"另外，把这三个供应商的合同调出来，我要仔细看一遍。"
"是。"
李主管离开后，沈知意靠在椅背上，闭上眼睛。
微表情洞察今日剩余2次，但她现在不需要用金手指。这件事已经很明显了——顾长渊出手了。
借王经理之手，切断项目供应链，让她刚谈成的合作胎死腹中。
好狠的招。
十五分钟后，会议室里坐满了人。
沈知意站在投影屏前，目光扫过在场的每一个人。技术部的陈明、财务部的李主管、还有几个核心项目的负责人都在。
"情况大家都知道了。"沈知意的声音平静，"三个核心供应商同时断供，项目面临停摆风险。我需要知道，以我们目前的库存，能撑多久？"
陈明举起手："技术物料大概能撑一周，但关键零部件只有三天的量。"
"财务方面呢？"沈知意看向李主管。
"如果项目延期，违约金是一个天文数字。"李主管推了推眼镜，"更麻烦的是，消息一旦传出去，股价肯定会受影响。"
会议室里一片沉默。
每个人都在等着沈知意的指示。
她沉默了几秒钟，然后开口："陈明，你联系技术部，把所有能替代的关键零部件列出来，不管国内国外，只要有货就行。"
"好。"
"李主管，你准备一份供应商违约的赔偿金计算，我要让他们知道，单方面毁约的代价。"
"明白。"
"其他人，该干什么干什么，不要让下面的人慌乱。"沈知意的目光坚定，"这只是商业上的博弈，我们能解决。"
散会后，沈知意独自留在会议室里。
她拿起手机，犹豫了一下，还是拨通了陆砚辞的电话。
"怎么，需要我了？"电话那头，陆砚辞的声音带着一丝慵懒的笑意。
"三个核心供应商同时断供，是顾长渊的手笔。"沈知意直截了当，"我需要知道陆氏那边的供应商能不能临时支援。"
"已经在联系了。"陆砚辞的声音变得认真，"给我两小时，我给你答复。"
"……谢谢。"
"谢什么。"陆砚辞轻笑一声，"你的事就是我的事。更何况，这本来也是陆氏的项目。"
挂断电话，沈知意看着窗外的乌云，眼神渐渐冷了下来。
顾长渊，既然你想玩，那我就陪你玩到底。
傍晚时分，天空终于下起了雨。
沈知意站在办公室窗前，看着雨滴打在玻璃上，划出一道道水痕。
手机响了，是陆砚辞发来的消息：
【陆氏的两家备用供应商同意临时供货，但价格比原来高15%。】
沈知意回复：【接受。另外，我要见见那三个毁约的供应商。】
【你想做什么？】
【让他们亲口告诉我，是谁在背后指使他们。】
消息发出去后，陆砚辞很久没有回复。
就在沈知意以为他不会回的时候，手机又震动了：
【明晚八点，澜韵会所。我安排。】
沈知意看着这条消息，嘴角微微上扬。
她转身走回办公桌前，从抽屉里拿出一份文件。
那是裴靳的调查报告，关于顾长渊与那家空壳公司的关联。
既然顾长渊选择正面交锋，那她也该亮出底牌了。
窗外的雨越下越大，雷声轰鸣。
暴风雨，真的来了。
