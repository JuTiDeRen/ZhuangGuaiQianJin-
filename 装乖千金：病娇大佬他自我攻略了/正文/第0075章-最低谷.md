# 第0075章 最低谷

消息传播得比想象中更快。

沈知意刚回到沈家，就发现自己被推上了风口浪尖。

各大媒体的头条都是关于她的"丑闻"——伪造的照片、编造的聊天记录、还有所谓"知情人士"的爆料。一时间，"沈家真千金私生活混乱"的话题冲上了热搜。

"姐姐，你没事吧？"沈清瑶假惺惺地走过来，脸上写满了"担忧"，"网上的那些消息……我相信都是假的，但其他人可能不会这么想。"

沈知意看着她，眼神冷得像冰。

"是吗？"她淡淡地说，"那你觉得，是谁放出去的呢？"

沈清瑶的表情僵了一下，随即恢复自然："我怎么会知道……"

"你不知道？"沈知意轻笑一声，"那U盘里的照片，又是谁准备的？"

沈清瑶的脸色瞬间惨白。

"你……你怎么……"

"我怎么会知道？"沈知意逼近一步，声音压得极低，"沈清瑶，你以为你做的那些事，真的没人知道吗？"

沈清瑶后退一步，眼中闪过一丝慌乱，但很快被怨毒取代。

"就算你知道又怎样？"她冷笑，"现在舆论已经发酵了，爸爸不会再让你出席慈善晚宴。没有晚宴，你拿什么跟我斗？"

她说完，转身离开，高跟鞋踩在大理石地面上发出刺耳的声响。

沈知意站在原地，看着她的背影，眼神幽深。

她回到房间，刚关上门，纪舒就推门而入。

"沈知意！"纪舒的声音带着压抑的愤怒，"你知不知道你干了什么？那些丑闻……你让我们沈家的脸往哪搁？"

"妈，"沈知意看着她，声音平静，"那些都是假的。"

"假的？"纪舒冷笑，"照片都出来了，你还说是假的？"

"照片可以伪造，聊天记录可以编造。"沈知意说，"你宁愿相信外人，也不愿意相信自己的女儿？"

纪舒愣了一下，但随即摇头："清瑶说得对，你在外面待了六年，谁知道你变成了什么样子……"

"六年。"沈知意重复着这两个字，声音里带着一丝苦涩，"是啊，六年。你们送走我的六年。"

她看着母亲，眼中最后一丝期待也熄灭了。

"妈，我问你最后一个问题。"她说，"如果六年前那场车祸不是意外，如果我是被人害的，你会怎么做？"

纪舒的表情变了变，但最终还是移开了视线："那都过去了……"

"过去？"沈知意笑了，那笑容凄凉而讽刺，"对我来说，那是一辈子的事。"

她走到窗边，背对着母亲："你走吧，我不想再谈了。"

纪舒张了张嘴，似乎想说什么，但最终只是叹了口气，转身离开。

房间里安静下来。

沈知意站在窗前，看着窗外的花园。阳光正好，但她的心却一片冰凉。

这就是她的母亲。

宁愿相信一个养女，也不愿意相信自己的亲生女儿。

手机响了，是陆砚辞打来的。

"我在楼下。"他的声音低沉，"下来。"

"我……"

"不管发生了什么，下来。"

他的语气不容置疑。

沈知意沉默了几秒，最终拿起外套下楼。

走出沈家大门，她看到陆砚辞靠在车门上，脸色阴沉得可怕。

"看到了？"她走过去，试图让自己的声音听起来轻松一些。

"看到了。"他直起身，目光灼灼地看着她，"你打算怎么办？"

"怎么办？"沈知意苦笑，"现在全网都在骂我，爸爸肯定不会再让我出席慈善晚宴。没有晚宴，我就没办法揭穿沈清瑶。"

她顿了顿，声音低了下来："这一局，我输了。"

陆砚辞盯着她看了几秒，然后做了一个让她意想不到的动作——

他伸手，将她拉入怀中。

"输？"他的声音在她头顶响起，"在我眼里，你永远不会输。"

"可是……"

"没有可是。"他松开她，双手捧住她的脸，"听着，小公主，这只是一时的困境，不是结局。"

"但慈善晚宴……"

"晚宴的事，我来想办法。"他的眼神坚定，"你现在要做的，就是相信我。"

沈知意看着他，看着那双眼睛里毫不掩饰的爱意和信任。

"为什么？"她问，"为什么要对我这么好？"

陆砚辞笑了，那笑容温柔得像春日的暖阳。

"因为我爱你啊。"他说，"从六年前就开始了。"

沈知意愣住了。

六年前？

"你……你说什么？"

陆砚辞没有回答，只是从口袋里掏出一样东西，放在她掌心。

是一张照片。

照片上的女孩穿着白色的连衣裙，站在樱花树下，笑得灿烂。

那是六年前的她。

"六年前，我在医院见过你。"陆砚辞的声音很轻，像是在回忆，"那时候你刚出车祸，浑身是伤，但眼神还是那么倔强。"

"我当时就想，这个女孩，我一定要保护她。"

沈知意的眼眶红了。

原来……原来他一直都在。

"所以，"陆砚辞擦去她眼角的泪水，"别放弃，好吗？"

"不管发生什么，我都在。"

沈知意看着他，良久，终于点了点头。

"好。"

她握紧手中的照片，眼神逐渐变得坚定。

不就是最低谷吗？

她从地狱里爬出来过一次，就能再爬第二次。

沈清瑶，你给我等着。

慈善晚宴上，我一定会让你付出代价。

回到房间，沈知意给音音打了电话。

"把所有证据都准备好，"她说，"慈善晚宴，我要一次性解决所有问题。"

"可是晚宴……"

"晚宴的事，陆砚辞会解决。"沈知意说，"我们只需要做好准备。"

挂断电话，她站在窗前，看着天边渐渐暗下来的天色。

暴风雨，就要来了。

而她，已经准备好了。

手机震动了一下，是林婉清发来的消息："明天的答复，你还打算给吗？"

沈知意看着屏幕，嘴角勾起一抹冷笑。

"给。"她回复，"但条件要改一改。"

"我要的不只是出席晚宴的资格。"

"我要的，是彻底的胜利。"

电话那头，林婉清回了一个字："好。"

窗外，最后一丝阳光消失在地平线。

黑夜，降临了。

但沈知意知道，黎明，不会太远了。
