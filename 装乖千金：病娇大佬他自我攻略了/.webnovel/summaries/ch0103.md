# 第0103章：睚眦必报2

## 章节摘要

**时间**：2026年7月22日上午

2. **绝不姑息**：面对三人的求情，沈知意坚持原则，指出背叛就是背叛，不因最后倒戈而改变。

4. **树立权威**：沈知意向父亲表明，她回来不是为了做老好人，而是要守护沈家。
